# SmartSchedule AI

## 🚀 Overview
SmartSchedule AI is an intelligent scheduling platform designed to help college students optimize their time, reduce stress, and improve productivity using AI.

## 💡 Problem
Students struggle with:
- Poor time management
- Overloaded schedules
- Lack of productivity insights

## ✅ Solution
Our AI:
- Automatically generates optimized schedules
- Detects burnout risk
- Tracks productivity in real-time
- Provides personalized recommendations

## 🔥 Features
- 🤖 AI schedule generator
- 📊 Live productivity dashboard
- 🎯 Goal tracking
- ⚠️ Burnout detection
- 🔐 Login + user profiles
- 👨‍💼 Admin dashboard (view all users)

## 🛠 Tech Stack
- Python (Tkinter GUI)
- AI logic (custom scheduling engine)
- Data tracking system
